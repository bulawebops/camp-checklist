# Stake Young Men Camp Checklist

## Camp Gear
- [ ] Sleeping bag
- [ ] Pillow
- [ ] Tent (if assigned)
- [ ] Flashlight or headlamp (with extra batteries)
- [ ] Reusable water bottle
- [ ] Backpack or duffel bag

## Clothing
- [ ] Modest camp clothes (for each day)
- [ ] Warm layer (hoodie/jacket)
- [ ] Rain jacket or poncho
- [ ] Extra socks
- [ ] Sleepwear
- [ ] Sturdy shoes
- [ ] Hat

## Toiletries & Health
- [ ] Toothbrush and toothpaste
- [ ] Soap and shampoo
- [ ] Deodorant
- [ ] Towel
- [ ] Sunscreen
- [ ] Bug spray
- [ ] Any personal medications

## Camp Essentials
- [ ] Scriptures
- [ ] Journal and pen
- [ ] Personal spending money (if needed)
- [ ] Permission/medical forms
- [ ] Assigned food or activity items (if requested)
